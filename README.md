# GetAmped_Optimize_Slanted_Run
优化因为垃圾键盘导致的GA无法起斜问题


# Usage
1、先双击运行install.bat安装驱动
	运行后会自动在C:\Windows\System32\drivers路径下会添加keyboard.sys 和mouse.sys两个文件
2、最好重启一下电脑
3、双击运行run.bat 【右键以记事本打开，可以修改按键缓冲数值和检查数值】
4、不想使用后，双击运行uninstall.bat以卸载驱动

# Principle
程序原理如下，能更好地帮你理解参数：
【程序目的】
检测方向键相邻方向键在短时间内被同时按下、松开、再按下，并将抖动优化
【输入参数】
参数1 delay_time 默认值0.010秒
参数2 check_time 默认值0.100秒
【程序逻辑】
1、初始时，没有键被延迟。假设当前按键事件为按下/松开(按下或松开)UP，如果在check_time内有松开/按下(与当前按键事件相反)UP以及松开/按下UP相邻按键，那么就将当前键放入延迟中。如果不满足条件，直接发送事件。
2、如果接受到新的按键事件，已经有按键被延迟，那么就判断新的按键事件是否和被延迟事件同为按下或者松开，再判断当前新按键在check_time之内是否被触发过。如果条件都满足，则将被延迟键和当前键一起发出，否则将当前按键事件 直接发出。
3、如果延迟键超时(超过delay_time)，则取消其延迟并发送。
